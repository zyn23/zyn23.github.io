How to optimise a windows install and make a windows custom ISO

Here is the guide that you should follow.

So... first we should start by creating and configuring our operative system... for a normal user I would recommend to just use the ISO that they recommend on the referenced site (Windows IoT LTSC Enterprise). Somehow if you are more of a power user I would recommend creating your own windows ISO using the guide. I also recommend running the Chris Titus Debloater (here) or just pasting this into the powershell "iwr -useb https://christitus.com/win | iex" if you don't want to use Chris Titus's Debloater you could follow the disabling Services part of the guide aswell as the Task Scheduler and Registry parts. I personnally cant be arsed to do the Group Policy Part manually so I don't do it and just use the Debloater + Task Scheduler optimizations.

I don't disable Windows Update neither do I disable Windows Defender... but YOU should follow their advice on these topics. (I make my Windows Updater just make security updates through the Debloater.)

I always use the software to debloat the NVIDIA and AMD drivers linked on the guide. (Yes I have 2 GPUs)
